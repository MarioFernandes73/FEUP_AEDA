Detailed Description

Classe Data. Tem como parâmetros um booleano (com valor default false e que verifica se a data está escrita numa forma curta - hora e minutos - ou longa - todos os parâmetros) e números inteiros correspondentes ao ano, mês, dia, hora e minuto.
